# Privacy Policy (Draft)

This platform collects user input for basic use and testing.

## 1. Data Collection (UK GDPR)

The platform only collects simple data:
- User input content  
- Time of submission  

It does not ask for names or sensitive personal data. This follows the idea of collecting only what is necessary.

## 2. Purpose of Data

The data is only used for:
- Running the platform  
- Simple analysis  
- Improving the system  

It will not be used for anything else.

## 3. Consent

Users must agree before submitting data.  
A checkbox is used to collect consent.

Users can also stop using the platform at any time.

## 4. Data Storage

Data is stored in a basic database.  
Access is limited and not public.

## 5. User Rights

Users can:
- Ask to see their data  
- Ask to delete their data  

---

# Design Changes for GDPR

To follow GDPR, I made these changes:

- Added a consent checkbox before submission  
- Users do not need to create an account  
- Added a button to delete data  

---

# Data Ownership and Privacy

Users own their own data.

The platform only stores and processes it.

Privacy is protected by:
- Collecting less data  
- Letting users delete data  
- Not sharing data with others
